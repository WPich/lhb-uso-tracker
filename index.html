สรุปภาพรวมตามสาขา

สาขาเครื่องทั้งหมดติดตั้งเสร็จกำลังติดตั้งยังไม่เริ่ม ผู้ใช้งานทั้งหมดผู้ใช้งานเสร็จ

Checklist มาตรฐาน (อ้างอิง)

Section 1 · User ปกติ (ก่อนติดตั้ง, ทำทุกคนที่ใช้เครื่อง):
ตรวจ USO Extension → Export รหัสผ่านจาก NetIQ → จดค่า AS400 → ปิด NetIQ → ทดสอบ Login ด้วยมือ

Section 2 · Admin (ติดตั้งเครื่อง, ทำครั้งเดียวต่อเครื่อง):
สลับ Admin → ตั้งค่า AS400/Printer + สร้าง Shortcut → Copy Icon ไป Startup → Uninstall NetIQ → Install USO Client → Reboot

Section 3 · User ปกติ (หลัง Reboot, ทำทุกคนที่ใช้เครื่อง):
สลับกลับ User → เปิด SSO Shortcut → ลงทะเบียนใน Self Service Portal → Import User/Password → กรอกรหัสผ่านครั้งแรกตามลำดับ (IBMLogin ก่อน) → ทดสอบ Login ผ่าน SSO → ให้ End User ยืนยันครั้งสุดท้าย

รายการแอป (อ้างอิง 14 แอป ไม่จำเป็นต้องใช้ครบทุกคน):
Fund Flow, Selling Agent, Internal Blacklist, BDS web, BDS Main, Fee Payment, AS400 (Prod), E-Form, IBAM, Pinpad Client, CMS (ATM), eKYC, Collection (New), FTP Initialise Parameter
