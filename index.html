Checklist มาตรฐาน (อ้างอิง)

Section 1 · User ปกติ (ก่อนติดตั้ง — ทำทุกคนที่ใช้เครื่อง)
ตรวจ USO Extension → Export รหัสผ่านจาก NetIQ → จดค่า AS400 → ปิด NetIQ → ทดสอบ Login ด้วยมือ

Section 2 · Admin (ทำครั้งเดียวต่อเครื่อง)
สลับ Admin → ตั้งค่า AS400/Printer + Shortcut → Copy Icon ไป Startup → Uninstall NetIQ → Install USO Client → Reboot
⚠️ ห้าม Uninstall NetIQ ก่อนเก็บรหัสผ่านครบ — เครื่องสาขาอาจมีผู้ใช้งานหลายคน ต้อง Export User/Password ของ "ทุกคน" ที่ใช้เครื่องนั้นให้เสร็จก่อน ถอนแล้วกู้คืนไม่ได้

Section 3 · User ปกติ (หลัง Reboot — ทำทุกคนที่ใช้เครื่อง)
สลับกลับ User → เปิด SSO Shortcut → ลงทะเบียน Self Service Portal → Import User/Password → กรอกรหัสครั้งแรก (IBMLogin ก่อน) → ทดสอบ Auto-login → End User ยืนยัน

แอปอ้างอิง 14 รายการ (ไม่จำเป็นต้องใช้ครบทุกคน)
Fund Flow, Selling Agent, Internal Blacklist, BDS web, BDS Main, Fee Payment, AS400 (Prod), E-Form, IBAM, Pinpad Client, CMS (ATM), eKYC, Collection (New), FTP Initialise Parameter
